# 🌸 Iris Flower Classification

A Machine Learning web application that predicts the species of an Iris flower based on its sepal and petal measurements.

## 🚀 Live Demo

[🔗 Add your Streamlit app link here](https://iris-flower-classification-fwacuqdcdcbynajv2z9xpu.streamlit.app/)

## 📌 About the Project

This project uses a Machine Learning classification model to classify Iris flowers into three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

The user can enter flower measurements through an interactive web interface and get the predicted flower species.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 🤖 Machine Learning Model

The model is trained on the Iris dataset and predicts the flower species based on:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
